# String Manipulation

Given this assignment:

`String s1 = "Red Apple";`

Select all expressions and compile and evaluate to true? 

## Correct

s1.toLowerCase().contains("app")

## Incorrect

s1.contains("app") 

> The method `contains` does a case-sensitive substring search

## Incorrect

s1.contains( "APP".toLowerCase() )

> The method `contains` does a case-sensitive substring search
